# Asset Credits

This prototype currently ships with no downloaded third-party art assets.

## Generated In Code

- Soccer ball: procedural Three.js sphere and wire overlay.
- Pitch: procedural planes, stripes, field lines, and penalty spot.
- Goal and net: procedural Three.js boxes and wireframe mesh.
- Player and keeper: procedural low-poly capsule, sphere, and box geometry.
- Lighting, camera, and materials: authored in project code.

## Approved Future Sources

When external assets are added, prefer licenses that allow public GitHub and LinkedIn demos:

- Kenney assets: CC0.
- Quaternius assets: CC0.
- KayKit assets: CC0 where marked by the source package.
- ambientCG materials: CC0.
- Poly Haven HDRIs/materials: CC0.

Avoid assets marked non-commercial, no-derivatives, or personal-use-only for this public demo.
